<h2>About</h2>
<p>Je construis des backends propres pour des produits réels: APIs, authentification, données, workers, temps réel et automatisations. Mon angle est simple: comprendre le besoin, poser une base claire, livrer quelque chose qui se maintient sans devenir fragile.</p>
<p><code>backend-lab --focus api,auth,data,workers,realtime --ship docker</code></p>
